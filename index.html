Project
Background

기존 운전면허학원은 체계적이지 않고 구시대적인 시스템, 강사의 문제와 부담스러운 가격 등의 문제점을 가지고 있습니다. 이러한
문제점은 높은 초보운전자 사고율로 직결됩니다. 그리하여 낙후된 운전 면허학원 시스템을 개선하고, 수강생 맞춤관리를 도와주는
'따고' 앱 서비스를 기획했습니다.

Project
Problem
운전면허 부실교육 문제

운전면허학원의 부실교육으로 인해
초보운전자 사고율이 36.4%로 매우 높습니다.

불법운전면허학원 多

불법운전면허학원의 피해 금액이
8억, 11억, 15억 등 매우 큽니다.

낙후된 운전면허학원 시스템

1. 수업 스케줄을 예약 / 변경할 때에 직접 안내
데스크에 가서 진행 해야 합니다.
2. 셔틀버스를 호출할 때에 하루 전날 직접 전화를
통해서 예약 해야 합니다.
3. 전반적인 시스템이 안내 데스크를 꼭 거쳐야 합니다.
4. 수업 후 자신의 점수와 부족한 점을 알기 어렵습니다.

강사 문제

1. 강사가 랜덤으로 배정되어 어떤 강사에게 수업을
받는지 알 수 없습니다.
2. 자신과 맞는 강사를 직접 선택할 수 없습니다.
3. 강사가 매번 바뀌어서 수업 스타일이 달라 혼란을
겪을 수 있습니다.

안내 데스크의 업무를 '따고'어플리케이션이 도와줌으로써, 안내데스크 직원들은 업무의 효율을 높일 수 있고 수강생들은 번거로운 절차를
간소화 시킬 수 있습니다.

Design
Design Keyword

Trust
Information
Simple
Young

따고는 신뢰감, 정보 전달, 심플함, 젊음 네 가지 키워드를 갖고 디자인하였습니다. 컬러는 신호등의 초록 불과 안전을 상징하는
녹색에 친근한 이미지를 주기 위해 트렌디함을 가미한 민트 컬러를 사용했습니다. 따고의 아이콘은 로고의 둥근 쉐입을 토대로
디자인하였으며, 면이 채워지지 않은 라인 아이콘을 사용하여 간결한 느낌의 앱 디자인과 어우러지도록 하였습니다.

전체적인 UI 요소들은 정보 안내/전달을 위해 깔끔하고 믿음이 가게
디자인하되, 젊은 타겟층을 위해 일러스트와 인터랙션으로 재미를
주었습니다.

Application
Function

맞춤형 프로세스를 제공받을 수 있습니다.

운전면허 취득과정에 맞추어 수강생이 해야 할 단계를 제시해주고 이행할 수 있게 도와줍니다.
사용자와 커뮤니케이션하는 듯한 느낌을 주고자 하였습니다. 사용자가 다음 과정을 직접 찾지 않아도 알아서 제시해 줍니다.
